Tailwind CSS 기본 사용법
Tailwind CSS는 유틸리티 클래스 기반의 CSS 프레임워크로, 빠르고 효율적으로 스타일을 적용할 수 있게 도와줍니다. 클래스 이름만으로 스타일을 지정하는 방식으로, 미리 정의된 클래스를 HTML 요소에 적용하여 디자인을 빠르게 구현할 수 있습니다. 설치 후 @tailwind 지시문을 사용하여 기본 스타일을 불러오고, tailwind.config.js 파일을 통해 커스터마이징도 가능합니다.
